But the addition to the embedding is x -> x + v_0^T R(mθ) e, where e is the basis. In homogeneous coordinates for the subspace (y, 1) where y is the coefficient of the position vector, the transformation is [[1, Δy], [0, 1]] with Δy proportional to sin(mθ - nθ) etc.

Specifically, if we have only one dimension of position encoding p(m) ∈ R, then x(m) = x_content(m) + p(m). Then x(m) - x(n) in the position component is p(m) - p(n). If p is sinusoidal, p(m) - p(n) = 2 cos((m+n)θ/2) sin(dθ/2). This depends on both d and m+n. So the shear parameter σ depends on d and also on the absolute center (m+n)/2. This is a "non-translation-invariant shear". So the transport matrix is N(σ(d; (m+n)/2)). Thus it is pure N, but with a parameter that depends on absolute position as well as d.

Thus mechanism 3 is indeed pure shear, but the shear parameter is not purely a function of d.

Now, for mechanism 4, the learned bias b(d) is content-dependent. The problem says "a separate projection produces a per-token, per-head feature that is combined with the distance d and added to the score." So we have v_m = W v x_m (or similar), and then b(m,n) = v_m^T φ(d) v_n? Or maybe b(d) = v_m · f(d). The exact form doesn't matter; it's learned.

Given this, the family of transport matrices is parameterized by v_m and v_n. Since v can vary continuously (or over a high-dimensional space), the set of matrices can cover regions in N, A, K.

Specifically, if f(d) has components that correspond to each generator, then v can weight them.

The obstruction: To be pinned to a single generator, the feature vector v must lie in a 1-dimensional subspace that aligns with the Lie algebra of that generator. But since W is a full-rank learned projection (typically), v generically spans a space of dimension equal to the head dimension. Thus it cannot be confined to one generator unless an explicit constraint is imposed. The Iwasawa uniqueness then forces any matrix with v in different directions to have different decompositions. Therefore, mechanism 4 necessarily spans more than one generator.

But is it "must it span more than one"? Not necessarily; if the projection is constrained (e.g., W outputs a scalar that only controls the shear parameter), then it can be pinned. But "by default" or "without constraint" it spans more. The problem asks "Can it be pinned to a single generator, or must it span more than one? State the obstruction to writing it as one generator."

So we answer: It can be pinned if the projection is rank-1 and aligned, but generically it must span more than one. The obstruction is the rank of the projection matrix and the uniqueness of Iwasawa decomposition: if two different content vectors produce matrices with different non-trivial factors in N, A, or K, they cannot both lie in the same single-generator subgroup.

But we need to be clear.

Now reasoning:

First principles: The convolution is a linear integral operator with kernel K(j-m). For |d| > R, the kernels don't overlap, so the contribution to q_m^T k_n from the convolution is zero (or constant). At |d| = R, the overlap is exactly one term. Thus the score as a function of d has a discontinuity in its (R+1)-th derivative unless b(d) compensates. The compatibility condition is that the Taylor expansion of b(d) around d = R must match that of the convolution contribution C(d) up to order R.

Specifically, if C(d) = Σ_{k=-R}^{R} c_k δ_{d,k}, then for |d| > R, C(d) = 0. Thus we need b(d) to approximate C(d) near the boundary to avoid a jump: b(R) = C(R), and for smoothness b^{(j)}(R) = C^{(j)}(R) for j = 1,...,L (where L is the smoothness order of b).

Pattern recognition: In practice, models like Longformer or BigBird combine local sliding window attention with global tokens and learned biases; the boundary artifacts are visible in attention maps.

Now, "Is there a residual the split cannot remove?"

First principles: The local convolution applies a linear map W_conv to the hidden state within the window. This map can have non-trivial rotation, dilation, and shear in feature space (e.g., if W_conv is not isotropic). The long-range bias b(d) is added to the scalar logit and cannot apply any linear transformation to q or k. Therefore, for |d| > R, the interaction between q_m and k_n lacks the linear-algebraic structure (the 2x2 transport with ρ,δ,σ) that is present for |d| ≤ R. Even with a learned content-dependent b, the term is v_m^T φ(d) v_n, which is bilinear in v but still adds only to the scalar score, not transforming q or k.

Thus the residual is structural: there is an irreducible "geometric gap" at long range. The split can match values and derivatives at d = R, but it cannot restore the missing feature-space rotation, dilation, or shear for |d| > R. This leaves a boundary layer where the model's geometric transformation changes abruptly from full SL2 to pure scalar.

We can also mention that because b(d) depends on content (v_m, v_n), it can adaptively approximate the missing linear transformation by adjusting v, but since v is the same for all pairs (m,n) with a given content, it cannot reproduce the position-specific linear transformation that the convolution applies.

Thus the answer: Yes, there is a residual; it is the missing 2x2 transport structure (ρ,δ,σ) for |d| > R, which a scalar bias can never fully replace.
